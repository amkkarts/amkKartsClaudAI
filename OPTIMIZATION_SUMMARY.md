## 📞 Quick Troubleshooting

**Pages don't load?**
- Check .htaccess isn't blocking requests
- Verify all CSS files uploaded
- Clear browser cache

**CSS not appearing?**
- Check file paths in `<link>` tags
- Verify CSS files in root directory
- Check browser DevTools (F12) for 404 errors

**Form not working?**
- Current form is JavaScript-based (no backend needed)
- Check script.js is loaded
- Open console (F12) for error messages
